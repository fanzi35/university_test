# Project Instructions

本项目是数学建模比赛项目。

## 项目资料

- 题目 PDF 放在 docs/ 中。
- Excel 附件如果只是参考格式或提交模板，放在 docs/reference_formats/ 中。
- 真正的原始数据才放在 data/raw/ 中。
- 处理后的数据放在 data/processed/ 中。
- 图表输出到 outputs/figures/。
- 表格结果输出到 outputs/tables/。

## 工作规则

- 开始任务前，先阅读 docs/ 中的题目要求。
- 不要把 docs/reference_formats/ 下的 Excel 当成原始数据。
- 不要覆盖任何题目附件和参考格式文件。
- 修改代码前，先说明准备修改哪些文件。
- 修改完成后，说明改了哪些文件、如何运行、如何验证结果。
- Python 代码要有简洁中文注释。
- 优先使用 numpy、pandas、matplotlib、scikit-learn、openpyxl。
